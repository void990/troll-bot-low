Инструкция:

Редактируем config.py

Добавляем API ID и API HASH (где и как регистрироваться знают все):

— nano config.py

api_id = ваш api_id

api_hash 'ваш api_hash'


Качаем требования что бы бот смог запустится:

— pip3 install -r requirements.txt
